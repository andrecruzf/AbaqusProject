\chapter*{Preface}

This report presents the results of a semester project conducted at the Chair of Artificial Intelligence in Mechanics and Manufacturing, ETH Zürich. The project is part of a broader research initiative on robust Forming Limit Curve identification, combining experimental testing and numerical simulation. This work focuses specifically on Task III of the project outline: the development of a fully automated numerical simulation framework for the generation of Forming Limit Curves using Nakazima and Marciniak test geometries in Abaqus/Explicit.

\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This work presents an automated finite element simulation framework for the numerical determination of Forming Limit Curves (FLCs) from Nakazima and Marciniak tests. The FE models are built in Abaqus/Explicit using a VUMAT user material subroutine implementing the Hosford-Coulomb fracture criterion. A range of specimen geometries (widths W20 to W200\,mm) are simulated for both test configurations to cover the full spectrum of strain paths from uniaxial tension to equibiaxial stretching. The critical major and minor principal strains at the onset of failure are extracted automatically from the output database and assembled into the FLC. The pipeline runs end-to-end on the ETH Euler HPC cluster with a single submission command, enabling systematic parametric studies of the influence of specimen geometry, sheet thickness, and test configuration on the forming limit.
