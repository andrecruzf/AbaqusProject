\chapter*{Preface}

This report presents the results of a semester project conducted at the Chair of Artificial Intelligence in Mechanics and Manufacturing, ETH Zürich, under the supervision of Prof.\ V.\ Grolleau and Dr.\ C.\ Roth. The project is part of a broader research initiative titled \textit{Towards Robust Forming Limit Curve Identification: Experiments and Modeling}, which combines experimental testing (Task~I), numerical identification criteria (Task~II), and automated simulation (Task~III). This work addresses Task~III: the development of a fully automated numerical simulation framework for the generation of Forming Limit Curves using Nakazima, Marciniak, and Punch-in-Punch test geometries in Abaqus/Explicit, deployed on the ETH Euler HPC cluster.

\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This work presents a fully automated finite element simulation framework for the numerical determination of Forming Limit Curves (FLCs) using three test configurations: Nakazima (hemispherical punch), Marciniak (flat punch), and Punch-in-Punch (PiP, two coaxial punches). The FE models are built in Abaqus/Explicit with a VUMAT user material subroutine implementing the Hosford-Coulomb fracture criterion. Seven specimen widths ($W \in \{20, 50, 80, 90, 100, 120, 200\}$\,mm) are simulated per configuration to cover the full strain path spectrum from uniaxial tension to equibiaxial stretching. Critical major and minor principal strains at the onset of failure are extracted automatically from the output database using a dome-zone filter consistent with ISO~12004-2, and assembled into the FLC. The pipeline runs end-to-end on the ETH Euler HPC cluster from a single deployment command, enabling systematic parametric studies of the influence of specimen geometry, sheet thickness, material orientation, and test configuration on the forming limit. Quasi-static validity is verified post-hoc by monitoring the kinetic-to-internal energy ratio throughout each simulation.
