\chapter{Conclusion}\label{chap:conclusion}

\section{Summary}

This project developed a fully automated pipeline for the numerical determination of Forming Limit Curves using Nakazima test simulations in Abaqus/Explicit. The pipeline covers the complete workflow from model building to FLC plotting and runs end-to-end on the ETH Euler HPC cluster. The Hosford-Coulomb VUMAT subroutine was successfully integrated and validated against reference results.

\section{Outlook}

Possible extensions of this work include:
\begin{itemize}
  \item Calibration of the HC material constants against experimental tensile and shear test data.
  \item Sensitivity analysis of the FLC with respect to friction coefficient and punch velocity.
  \item Comparison of numerical FLC with experimental Nakazima test results.
  \item Extension to the Marciniak test geometry for additional strain path coverage.
\end{itemize}
